Apunte Teórico de Conceptos Utilizados en los Ejemplos de C

1. Entrada y Salida de Datos

  • printf: Imprime texto o variables en pantalla.
    Sintaxis: printf("Texto %d", variable);
  • scanf: Lee datos desde teclado (para enteros, cadenas, etc.).
    Sintaxis: scanf("%d", &variable);
    Para cadenas: scanf("%s", cadena);
  • fgets: Lee una línea de texto (incluye espacios).
    Sintaxis: fgets(cadena, tamaño, stdin);

2. Declaración y Uso de Variables

  • int, char, float, etc.: Tipos de datos básicos.
    Ejemplo: int numero; char letra;
  • Arreglos (vectores): Colección de elementos del mismo tipo.
    Ejemplo: int arreglo[10]; char cadena[100];

3. Inicialización y Recorrido de Arreglos

  • Inicialización manual: 
    int arreglo[3] = {1, 2, 3};
  • Recorrido con for: 
    for(int i = 0; i < tamaño; i++) {
    // usar arreglo[i]
}

4. Funciones

  • Declaración: 
    tipo nombre_funcion(parámetros) {
    // código
    return valor;
}
  • Ejemplo: 
    int suma(int a, int b) {
    return a + b;
}

5. Uso de srand y rand para Números Aleatorios

  • srand(time(NULL));: Inicializa la semilla aleatoria.
  • rand();: Devuelve un número pseudoaleatorio.
  • Para un rango: rand() % rango + inicio;
  • Ejemplo: 
    int n = rand() % 10 + 1; // Entre 1 y 10

6. Cadenas de Caracteres (Strings)

  • Declaración: char cadena[100];
  • Lectura: fgets(cadena, 100, stdin);
  • Eliminación de salto de línea: 
    cadena[strcspn(cadena, "\n")] = '\0';
  • Concatenar: strcat(cadena1, cadena2);
  • Comparar: strcmp(cadena1, cadena2);
  • Buscar subcadena: strstr(cadena, subcadena);

7. Funciones de la Biblioteca <string.h>

  • strlen(cadena): Devuelve la longitud.
  • strcpy(destino, origen): Copia cadenas.
  • strcat(destino, origen): Concatena cadenas.
  • strcmp(cad1, cad2): Compara cadenas.
  • strstr(cad, sub): Busca subcadena.

8. Funciones de la Biblioteca <ctype.h>

  • toupper(c): Convierte a mayúscula.
  • tolower(c): Convierte a minúscula.
  • isalpha(c): ¿Es letra?
  • isdigit(c): ¿Es dígito?

9. Matrices (Arreglos Bidimensionales)

  • Declaración: int matriz[3][3];
  • Acceso: matriz[fila][columna];

10. Palíndromos

  • Comparar caracteres desde los extremos hacia el centro.
  • Usar tolower para ignorar mayúsculas/minúsculas.

11. Contar Frecuencias

  • Usar un vector de frecuencias.
    Ejemplo: 
    int frecuencia[51] = {0};
frecuencia[valor]++;

12. Comparación de Cadenas Ignorando Mayúsculas/Minúsculas

  • Convertir ambas cadenas a minúsculas o mayúsculas antes de comparar.

13. Operaciones con Vectores y Matrices

  • Suma de elementos, producto, combinación, ordenamiento (por ejemplo, método de intercambio).

14. Sintaxis Básica de un Programa en C

#include <stdio.h>
int main() {
    // código
    return 0;
}

15. Estructuras de Control

  • if/else con condiciones simples y compuestas: 
    if (nota >= 6 && asistencia >= 75) {
    printf("Aprobado.\n");
} else {
    printf("Desaprobado.\n");
}
  • for: 
    for (inicialización; condición; incremento) {
    // código
}

16. Ejemplo de Uso de Funciones y Arreglos

int sumaVector(int v[], int n) {
    int suma = 0;
    for (int i = 0; i < n; i++) {
        suma += v[i];
    }
    return suma;
}

17. Ejemplo de Conversión de Temperaturas

float celsius = (fahrenheit - 32) * 5.0 / 9.0;

18. Ejemplo de Verificación de Número Primo

bool esPrimo(int n) {
    if (n <= 1) return false;
    for (int i = 2; i*i <= n; i++) {
        if (n % i == 0) return false;
    }
    return true;
}

19. Ordenamiento Simple (Burbuja)

for (int i = 0; i < n-1; i++) {
    for (int j = 0; j < n-i-1; j++) {
        if (v[j] > v[j+1]) {
            int temp = v[j];
            v[j] = v[j+1];
            v[j+1] = temp;
        }
    }
}

20. Observaciones Generales

  • Siempre inicializa los arreglos y variables.
  • Usa funciones para dividir el código en partes reutilizables.
  • Usa comentarios para aclarar el propósito del código.

21. Condiciones Compuestas (Dobles Condiciones)

  • Se utilizan operadores lógicos para combinar condiciones.
  • AND (&&): Ambas condiciones deben ser verdaderas.
  • OR (||): Al menos una condición debe ser verdadera.
  • NOT (!): Niega una condición (verdadero → falso).
  • Ejemplos: 
    if (edad >= 18 && edad <= 65) {
    printf("Edad válida para trabajar.\n");
}

if (nota < 4 || asistencia < 75) {
    printf("Alumno desaprobado.\n");
}

if (!esValido) {
    printf("Valor no válido.\n");
}
